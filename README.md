# 个人学习与转型日志 / Personal Learning & Transition Log

## 👋 欢迎 / Welcome
This repository serves as a public journal documenting my intellectual transition from a background in Law to Computer Science. It tracks my progress in structural thinking, programming mastery, and cognitive maintenance at age 43.

*本仓库是一个公开日志，记录了我从法律背景向计算机科学转型的过程。它主要跟踪我在结构化思维、编程精通度以及 43 岁时的认知维护方面的进展。*

---

## 📚 仓库内容与焦点 / Key Focus Areas

本仓库主要包含以下几个学习模块的深度记录：

1.  **Java & OOP 精通:** 核心编程概念与面向对象编程（OOP）原理的掌握。
2.  **软件测试方法论 (Software Testing):** 深入研究 Statement Coverage (SC)、Branch Coverage (BC)、等价类划分（EP）和边界值分析（BVA）等白盒测试技术。
3.  **算法与数据结构 (Algorithms & DS):** LeetCode 刷题记录、解题思路和性能优化反思。
4.  **学习哲学:** 关于加速学习、知识杠杆、以及通过持续学习抵抗精神衰退的思考。

## 🧭 仓库结构 / Repository Structure Guide
* `posts/`: 所有的日记和学习反思文章，以 Markdown 格式存储。
* `code/`: 对应的 Java 代码片段、测试用例和算法实现。
* `config/`: 自动化和环境配置相关的辅助文件。
